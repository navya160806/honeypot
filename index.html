About The Project

The MCP Self-Auditing AI Model Monitor is an intelligent AI system designed to automatically monitor machine learning models in production. The system analyzes prediction behavior, detects bias and data drift, calculates model risk scores, and generates explainable reports using Google Gemini AI. It can also repair models by cleaning data and retraining them automatically.
